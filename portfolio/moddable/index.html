What it is

An open-source tabletop and digital games studio shipping at startup speed. Everything is built using AI-augmented engineering workflows — LLM-driven development as core methodology, enabling solo-founder velocity across 10+ concurrent repositories.

What shipped

  • Dungeon Chess — asymmetric skirmish game with custom heuristic-driven AI opponent (dungeon.moddable.games)
  • Moddable Chess Engine — 64 playable variants, minimax AI, embed API, plugin architecture (chess.moddable.games)
  • Moddable Hexmaps — seeded procedural map generator with JSON export and URL param API (hex.moddable.games)
  • Nukes, Mongo, Endless Skies — tabletop games with digital tools and rulebooks
  • PDF Pagination Engine — custom document assembly producing 190+ page rulebook libraries with orphan prevention
  • Developer documentation — "Build a Variant" guide, embed APIs, structured data interfaces
  • Multi-audience presentation system — 7 audience variants from one master deck

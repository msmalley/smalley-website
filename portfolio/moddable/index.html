What it is

An open-source tabletop and digital games studio shipping at startup speed. Everything is built using AI-augmented engineering workflows — LLM-driven development as core methodology, enabling solo-founder velocity across 10+ concurrent repositories.

What shipped

  • Moddable Chess Engine (v0.7.0) — evolved from a 70-variant chess app into a consumer game framework with reusable game controller, renderer extension hooks, replay API, unit templates, and terrain predicates. Powers two games from the same core (chess.moddable.games)
  • Dungeon Chess — asymmetric skirmish game running entirely on MCE as a game subsystem. Custom renderer hooks, faction-based AI, XP-budgeted drafting — all built on top of the chess engine's consumer SDK (dungeon.moddable.games)
  • Moddable Hexmaps — seeded procedural map generator with JSON export and URL param API (hex.moddable.games)
  • Nukes, Mongo, Endless Skies — tabletop games with digital tools and rulebooks
  • PDF Pagination Engine — custom document assembly producing 190+ page rulebook libraries with orphan prevention
  • Developer documentation — tiered guides (Build a Variant, Crazyhouse, Poison Chess), consumer integration guide (Building Dungeon Chess), full API reference
  • Multi-audience presentation system — 7 audience variants from one master deck

Why it matters

Demonstrates the ability to ship production-quality software at unprecedented velocity and then evolve it into framework-level architecture. The chess engine didn't stay as a standalone app — it became a consumer SDK that powers a completely different game (Dungeon Chess) through renderer hooks, game controllers, and unit templates. Every project is MIT-licensed, publicly accessible, with proper documentation, version control, and deployment pipelines.
